Messages

La messagerie Twake permet d'envoyer du texte stylisé à l'aide de commandes textes.

Commandes de base

**gras**
--barré--
~~italique~~
<<monospace>>
__souligné__

Commandes avancées

<<<javascript function a(){} >>>
<<<php <?php function a(){} >>>
quote{Title}{Message}
color{red}{Texte en rouge}
color{#F08777}{Texte coloré}
#F00EE9 (affiche la couleur automatiquement)
@username
:emoji_name: (ajoute l'emoji nommé emoji_name)

Commandes externes

\say [fr] Bonjour ! (prononce chez tous les utilisateurs)
\signal (envoi un signal lumineux)
